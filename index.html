Introduction

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It allows you to control the layout, colors, fonts, and overall appearance of web pages.

This webpage is a demonstration of the different basic styles that can be applied to HTML through CSS, separating content from design, making it easier to maintain and update a website like this one.

The CSS file used to style the example are included in the `<link>` tag within the `<head><∖head>` of this HTML file.

The Box Model

Each inline and block element is implemented using a box model consisting of a content area, padding and margin.

In the box model, the content is surrounded by padding, which is then surrounded by a border, and finally, there is a margin outside the border.

Texts and Fonts
